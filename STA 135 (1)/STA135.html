I. Introduction

The composition of glass involves various chemicals, and the type of glass changes depending on the proportions of these substances. There are many different types of glass, such as those used for car windows, headlamps, cookware, and more. Each type has its unique chemical composition, making it possible to classify them using multivariate analysis. Successfully classifying the type of glass based on these chemicals could enable us to apply similar techniques to other fields in the future. In this project, we aim to classify the types of glass based on the proportions of different chemicals using the glass dataset from the mlbench package. To achieve this, we will adjust the data, such as dimensions reduction, and employ basic supervised learning techniques such as Linear Discriminant Analysis (LDA) and Quadratic Discriminant Analysis (QDA).

II. Exploratory Data Analysis

The glass dataset used in this project consists of a total sample size of 214. It contains information on nine chemical components: Sodium (Na), Magnesium (Mg), Aluminum (Al), Silicon (Si), Potassium (K), Calcium (Ca), Barium (Ba), and Iron (Fe). The values for these components are measures of the bending of a ray of light. The following illustrates the distribution of each chemical component.

III. Methodology

This project involves classifying glass types using chemical data, which requires careful data preprocessing due to the large number of variables compared to the sample size. Initially, glass types 3, 5, and 7 were combined into a single class to balance the sample sizes with those of glass types 1 and 2, resulting in approximately 70 samples per class out of a total of 214 samples.

III. 1. Dimension Reduction

Given the 9 features in the dataset, Principal Component Analysis (PCA) was employed for dimension reduction to extract essential information and facilitate efficient modeling. PCA involves calculating the eigenvalues and corresponding eigenvectors of the data to determine the principal components based on the amount of information each eigenvalue represents. Specifically, the covariance matrix of the dataset is computed, followed by its eigenvalues and eigenvectors, which are then sorted in descending order of the eigenvalues. The principal components are the eigenvectors corresponding to the largest eigenvalues. This process was visualized using a scree plot to determine the number of principal components needed to capture most of the variance or information in the data.

III. 2. Classification Method & Modeling

After selecting the appropriate number of principal components, the dataset was transformed accordingly, and these new components were used as features for the classification model. A classification model was then constructed using Linear Discriminant Analysis (LDA) or Quadratic Discriminant Analysis (QDA), depending on the equality of covariances tested using Box's M Test.

The final model's performance was evaluated using k-fold cross-validation, ensuring robustness and generalizability. This project used five folds (K = 5). Next, Box's M Test was applied to verify the equality of covariances across classes. The test result showed a p-value < 2.2e-16, which indicates a rejection of the null hypothesis of equal covariances. As a result, Quadratic Discriminant Analysis (QDA) was selected as the appropriate classification model for this dataset.

The data was split into training and test sets in a 60:40 ratio. QDA was applied to the training set, and the model was then validated on the test set, resulting in an accuracy of 0.622. To further evaluate the model's performance, K-Fold cross-validation (K=5) was performed, yielding a classification rate of 0.60.

V. Conclusion

In this project, we processed the data as needed to perform efficient and accurate classification. First, we analyzed the distribution of the data. Then, to handle the high-dimensional nature of the data relative to the number of samples, we reduced the nine covariates to four principal components. This was done to retain as much variability as possible while simplifying the dataset. Based on this processed data, we constructed a Quadratic Discriminant Analysis model. Despite these efforts, the classification accuracy on the test set was around 62.2%, and the 5-fold cross-validation yielded a classification rate of 60%. The confusion matrix revealed that misclassifications were common for particular glass types, indicating that the features used might not be sufficiently discriminative. The classification rate of around 60 percent can be disappointing. This may be due to the normality issue mentioned earlier. Linear Discriminant Analysis (LDA) and Quadratic Discriminant Analysis (QDA) rely on the normality assumption. However, the distribution of features violated this assumption. Although this project focuses on generative cases like LDA and QDA, to improve performance, alternative methods for building classification models or additional data transformations using other methods, such as t-SNE, are recommended.
